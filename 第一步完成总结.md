## 📝 设计说明

### 设计选择

1. **工具函数命名空间模式**（参考模板）
   - 使用 `lms::core::pathUtils` 命名空间
   - 函数式风格，无需实例化
   - 简单直接，易于使用

2. **使用 std::filesystem**
   - C++17 标准库
   - 跨平台支持
   - 类型安全

3. **错误处理**
   - 使用 `std::error_code` 避免异常
   - 静默处理错误，返回 false
