## 🚀 快速开始 / Быстрый старт

### 启动服务器 / Запуск сервера

```bash
# 1. 进入构建目录
# 1. Переход в каталог сборки
cd build

# 2. 运行服务器
# 2. Запуск сервера
./src/lms/lms ../conf/lms.conf

# 3. 在浏览器中访问
# 3. Доступ в браузере
# http://localhost:5082
```

### 首次登录 / Первый вход

1. **打开浏览器**，访问 `http://localhost:5082`
2. **初始化向导**会自动显示，引导你创建第一个管理员账户
3. **填写信息**：
   - 用户名（建议使用 `admin`）
   - 密码（请设置强密码）
   - 确认密码
4. **完成设置**后，系统会自动跳转到主界面

1. **Откройте браузер** и перейдите на `http://localhost:5082`
2. **Мастер инициализации** автоматически отобразится для создания первой учётной записи администратора
3. **Заполните информацию**:
   - Имя пользователя (рекомендуется `admin`)
   - Пароль (установите надёжный пароль)
   - Подтверждение пароля
4. После **завершения настройки** система автоматически перейдёт на главный интерфейс

## ⚙️ 首次设置 / Первоначальная настройка

### 添加媒体库 / Добавление медиатеки

**作为管理员，你需要先添加音乐库才能开始使用：**

**Как администратор, вам нужно сначала добавить музыкальную библиотеку:**

#### 步骤 1：进入媒体库管理 / Шаг 1: Переход в управление медиатеками

1. 点击顶部导航栏的 **"媒体库"** 链接（仅管理员可见）
2. 或直接访问：`http://localhost:5082/admin/libraries`

1. Нажмите на ссылку **"Медиатеки"** в верхней панели навигации (видно только администратору)
2. Или перейдите напрямую: `http://localhost:5082/admin/libraries`

#### 步骤 2：添加新媒体库 / Шаг 2: Добавление новой медиатеки

1. 点击 **"添加媒体库"** 按钮
2. 填写媒体库信息：
   - **名称**：给媒体库起个名字（如"我的音乐"）
   - **路径**：音乐文件所在的目录路径
     - 示例：`/mnt/e/Music` 或 `/home/user/Music`
     - **注意**：确保路径存在且可读
3. 点击 **"保存"**

1. Нажмите кнопку **"Добавить медиатеку"**
2. Заполните информацию о медиатеке:
   - **Имя**: дайте имя медиатеке (например, "Моя музыка")
   - **Путь**: путь к каталогу с музыкальными файлами
     - Пример: `/mnt/e/Music` или `/home/user/Music`
     - **Примечание**: убедитесь, что путь существует и доступен для чтения
3. Нажмите **"Сохранить"**

#### 步骤 3：启动扫描 / Шаг 3: Запуск сканирования

1. 添加媒体库后，系统会自动开始扫描
2. 你也可以手动触发扫描：
   - 进入 **"扫描器"** 页面（`/admin/scanner`）
   - 点击 **"开始扫描"** 按钮
3. **扫描过程**：
   - 系统会递归扫描媒体库目录
   - 提取音频文件的元数据（艺术家、专辑、曲目信息等）
   - 提取封面图片
   - 生成聚类标签（流派、年份等）
   - 这个过程可能需要几分钟到几十分钟，取决于音乐库大小

1. После добавления медиатеки система автоматически начнёт сканирование
2. Вы также можете запустить сканирование вручную:
   - Перейдите на страницу **"Сканер"** (`/admin/scanner`)
   - Нажмите кнопку **"Начать сканирование"**
3. **Процесс сканирования**:
   - Система рекурсивно сканирует каталог медиатеки
   - Извлекает метаданные аудиофайлов (исполнитель, альбом, информация о треке и т.д.)
   - Извлекает изображения обложек
   - Генерирует метки кластеров (жанр, год и т.д.)
   - Этот процесс может занять от нескольких минут до нескольких десятков минут в зависимости от размера музыкальной библиотеки

#### 步骤 4：等待扫描完成 / Шаг 4: Ожидание завершения сканирования

- 扫描进度会显示在扫描器页面
- 完成后，你可以在主界面看到你的音乐了！

- Прогресс сканирования отображается на странице сканера
- После завершения вы сможете увидеть свою музыку на главном интерфейсе!

## 🎵 浏览音乐 / Просмотр музыки

### 主界面导航 / Навигация главного интерфейса

LMS 提供多种浏览音乐的方式：

LMS предоставляет различные способы просмотра музыки:

#### 1. 艺术家视图 / Представление исполнителей

- **访问**：点击导航栏的 **"艺术家"** 或访问 `/artists`
- **功能**：
  - 显示所有艺术家列表
  - 支持按艺术家类型过滤（主艺术家、客串艺术家等）
  - 支持搜索艺术家
  - 点击艺术家名称查看该艺术家的所有专辑

- **Доступ**: нажмите **"Исполнители"** в панели навигации или перейдите на `/artists`
- **Функции**:
  - Отображает список всех исполнителей
  - Поддерживает фильтрацию по типу исполнителя (основной, приглашённый и т.д.)
  - Поддерживает поиск исполнителей
  - Нажмите на имя исполнителя, чтобы просмотреть все его альбомы

#### 2. 专辑视图 / Представление альбомов

- **访问**：点击导航栏的 **"专辑"** 或访问 `/releases`
- **功能**：
  - 以网格形式显示所有专辑封面
  - 支持多种排序方式（随机、按名称、按日期等）
  - 支持搜索专辑
  - 点击专辑封面查看专辑详情和曲目列表

- **Доступ**: нажмите **"Альбомы"** в панели навигации или перейдите на `/releases`
- **Функции**:
  - Отображает все обложки альбомов в виде сетки
  - Поддерживает различные способы сортировки (случайный, по имени, по дате и т.д.)
  - Поддерживает поиск альбомов
  - Нажмите на обложку альбома, чтобы просмотреть детали альбома и список треков

#### 3. 曲目视图 / Представление треков

- **访问**：点击导航栏的 **"曲目"** 或访问 `/tracks`
- **功能**：
  - 显示所有曲目列表
  - 显示详细信息（艺术家、专辑、时长等）
  - 支持多种排序方式
  - 支持搜索曲目

- **Доступ**: нажмите **"Треки"** в панели навигации или перейдите на `/tracks`
- **Функции**:
  - Отображает список всех треков
  - Отображает подробную информацию (исполнитель, альбом, длительность и т.д.)
  - Поддерживает различные способы сортировки
  - Поддерживает поиск треков

#### 4. 播放列表视图 / Представление плейлистов

- **访问**：点击导航栏的 **"播放列表"** 或访问 `/tracklists`
- **功能**：
  - 显示所有播放列表
  - 创建新播放列表
  - 编辑现有播放列表
  - 删除播放列表

- **Доступ**: нажмите **"Плейлисты"** в панели навигации или перейдите на `/tracklists`
- **Функции**:
  - Отображает все плейлисты
  - Создание нового плейлиста
  - Редактирование существующего плейлиста
  - Удаление плейлиста

### 使用过滤器 / Использование фильтров

LMS 提供强大的过滤功能，帮助你快速找到想要的音乐：

LMS предоставляет мощную функцию фильтрации для быстрого поиска нужной музыки:

#### 添加过滤器 / Добавление фильтров

1. 点击导航栏右侧的 **过滤器图标**（漏斗图标）
2. 在弹出对话框中选择过滤条件：
   - **聚类标签**：流派、年份、国家、语言等
   - **标签**：自定义标签
   - **媒体库**：选择特定媒体库
   - **专辑类型**：专辑、单曲、EP 等
3. 点击 **"应用"** 应用过滤器

1. Нажмите на **иконку фильтра** (иконка воронки) справа в панели навигации
2. Выберите условия фильтрации во всплывающем диалоге:
   - **Метки кластеров**: жанр, год, страна, язык и т.д.
   - **Метки**: пользовательские метки
   - **Медиатека**: выбор конкретной медиатеки
   - **Тип альбома**: альбом, сингл, EP и т.д.
3. Нажмите **"Применить"** для применения фильтров

#### 管理活动过滤器 / Управление активными фильтрами

- 活动过滤器会显示在导航栏下方
- 点击过滤器标签上的 **"×"** 可以删除该过滤器
- 点击 **"清除所有"** 可以清除所有过滤器

- Активные фильтры отображаются под панелью навигации
- Нажмите **"×"** на метке фильтра, чтобы удалить его
- Нажмите **"Очистить всё"** для удаления всех фильтров

## 🎧 播放音乐 / Воспроизведение музыки

### 播放控制 / Управление воспроизведением

#### 基本播放操作 / Основные операции воспроизведения

1. **播放单个曲目**：
   - 在曲目列表中，点击曲目名称或播放按钮
   - 或右键点击曲目，选择 **"播放"**

2. **播放专辑**：
   - 进入专辑详情页
   - 点击 **"播放全部"** 按钮
   - 或右键点击专辑封面，选择播放方式

3. **播放艺术家所有曲目**：
   - 进入艺术家页面
   - 点击 **"播放全部"** 按钮

1. **Воспроизведение одного трека**:
   - В списке треков нажмите на имя трека или кнопку воспроизведения
   - Или щёлкните правой кнопкой мыши на треке и выберите **"Воспроизвести"**

2. **Воспроизведение альбома**:
   - Перейдите на страницу деталей альбома
   - Нажмите кнопку **"Воспроизвести всё"**
   - Или щёлкните правой кнопкой мыши на обложке альбома и выберите способ воспроизведения

3. **Воспроизведение всех треков исполнителя**:
   - Перейдите на страницу исполнителя
   - Нажмите кнопку **"Воспроизвести всё"**

#### 播放方式选择 / Выбор способа воспроизведения

右键点击音乐项时，可以选择不同的播放方式：

При щелчке правой кнопкой мыши на музыкальном элементе можно выбрать различные способы воспроизведения:

- **播放**：立即播放，清空当前队列
- **下一首播放**：插入到当前播放位置之后
- **追加到末尾**：添加到队列末尾
- **随机播放**：随机播放所有曲目

- **Воспроизвести**: немедленное воспроизведение, очищает текущую очередь
- **Воспроизвести следующим**: вставляет после текущей позиции воспроизведения
- **Добавить в конец**: добавляет в конец очереди
- **Случайное воспроизведение**: случайное воспроизведение всех треков

### 播放器控件 / Элементы управления плеером

页面底部的播放器提供以下功能：

Плеер в нижней части страницы предоставляет следующие функции:

- **播放/暂停**：控制播放状态
- **上一首/下一首**：切换曲目
- **进度条**：显示播放进度，可以拖拽跳转
- **音量控制**：调整播放音量
- **当前曲目信息**：显示正在播放的曲目、艺术家和专辑

- **Воспроизведение/пауза**: управление состоянием воспроизведения
- **Предыдущий/следующий**: переключение треков
- **Полоса прогресса**: отображает прогресс воспроизведения, можно перетаскивать для перехода
- **Управление громкостью**: регулировка громкости воспроизведения
- **Информация о текущем треке**: отображает воспроизводимый трек, исполнителя и альбом

## 📋 播放队列 / Очередь воспроизведения

### 查看播放队列 / Просмотр очереди воспроизведения

- **访问**：点击导航栏的 **"播放队列"** 或访问 `/playqueue`
- **功能**：
  - 查看当前队列中的所有曲目
  - 查看队列统计信息（曲目数量、总时长）
  - 管理队列中的曲目

- **Доступ**: нажмите **"Очередь воспроизведения"** в панели навигации или перейдите на `/playqueue`
- **Функции**:
  - Просмотр всех треков в текущей очереди
  - Просмотр статистики очереди (количество треков, общая длительность)
  - Управление треками в очереди

### 队列操作 / Операции с очередью

#### 添加曲目到队列 / Добавление треков в очередь

1. **从浏览页面添加**：
   - 右键点击曲目、专辑或艺术家
   - 选择 **"下一首播放"** 或 **"追加到末尾"**

2. **从播放队列页面添加**：
   - 点击 **"添加曲目"** 按钮
   - 搜索并选择要添加的曲目

1. **Добавление со страницы просмотра**:
   - Щёлкните правой кнопкой мыши на треке, альбоме или исполнителе
   - Выберите **"Воспроизвести следующим"** или **"Добавить в конец"**

2. **Добавление со страницы очереди воспроизведения**:
   - Нажмите кнопку **"Добавить трек"**
   - Найдите и выберите треки для добавления

#### 队列管理功能 / Функции управления очередью

- **清空队列**：点击 **"清空"** 按钮移除所有曲目
- **随机播放**：点击 **"随机"** 按钮打乱队列顺序
- **删除曲目**：点击曲目旁边的 **"×"** 按钮删除单个曲目
- **保存为播放列表**：点击 **"保存"** 按钮将当前队列保存为播放列表

- **Очистка очереди**: нажмите кнопку **"Очистить"** для удаления всех треков
- **Случайное воспроизведение**: нажмите кнопку **"Случайно"** для перемешивания порядка очереди
- **Удаление трека**: нажмите кнопку **"×"** рядом с треком для удаления отдельного трека
- **Сохранение как плейлист**: нажмите кнопку **"Сохранить"** для сохранения текущей очереди как плейлиста

### 播放模式 / Режимы воспроизведения

#### 重复播放 / Повтор

- 点击播放队列中的 **"重复"** 复选框
- 启用后，队列会循环播放
- 当最后一首播放完后，会自动从第一首重新开始

- Нажмите чекбокс **"Повтор"** в очереди воспроизведения
- После включения очередь будет воспроизводиться циклически
- После воспроизведения последнего трека автоматически начнётся с первого

#### 电台模式 / Режим радио

- 点击播放队列中的 **"电台"** 复选框
- 启用后，系统会根据当前播放的音乐自动推荐并添加相似曲目
- 队列会持续自动填充，提供无限的音乐流

- Нажмите чекбокс **"Радио"** в очереди воспроизведения
- После включения система автоматически рекомендует и добавляет похожие треки на основе текущей воспроизводимой музыки
- Очередь будет постоянно автоматически пополняться, обеспечивая бесконечный поток музыки

## 🔍 搜索功能 / Функция поиска

### 基本搜索 / Базовый поиск

1. **使用搜索框**：
   - 在导航栏右侧找到搜索框
   - 输入关键词（艺术家名、专辑名、曲目名等）
   - 按 Enter 或点击搜索图标

2. **搜索结果**：
   - 系统会显示匹配的艺术家、专辑和曲目
   - 点击结果项可以查看详情或播放

1. **Использование поля поиска**:
   - Найдите поле поиска справа в панели навигации
   - Введите ключевое слово (имя исполнителя, название альбома, название трека и т.д.)
   - Нажмите Enter или щёлкните иконку поиска

2. **Результаты поиска**:
   - Система отобразит соответствующих исполнителей, альбомы и треки
   - Нажмите на элемент результата для просмотра деталей или воспроизведения

### 高级搜索技巧 / Продвинутые приёмы поиска

- **多关键词搜索**：使用空格分隔多个关键词
- **精确匹配**：使用引号进行精确匹配（如 `"The Beatles"`）
- **组合过滤**：搜索后可以继续使用过滤器进一步筛选

- **Поиск по нескольким ключевым словам**: используйте пробелы для разделения нескольких ключевых слов
- **Точное совпадение**: используйте кавычки для точного совпадения (например, `"The Beatles"`)
- **Комбинированная фильтрация**: после поиска можно продолжить использование фильтров для дальнейшей фильтрации

## 📝 播放列表 / Плейлисты

### 创建播放列表 / Создание плейлиста

#### 方法 1：从播放队列创建 / Способ 1: Создание из очереди воспроизведения

1. 在播放队列中添加你想要的曲目
2. 点击 **"保存"** 按钮
3. 输入播放列表名称
4. 选择保存方式：
   - **新建播放列表**：创建新的播放列表
   - **替换现有播放列表**：选择现有播放列表并替换其内容
5. 点击 **"确认"** 保存

### 播放器设置 / Настройки плеера

#### 转码设置 / Настройки транскодирования

- **转码模式**：
  - **从不**：不进行转码，直接传输原始文件
  - **总是**：总是转码为指定格式
  - **格式不支持时**：仅在浏览器不支持原始格式时转码（推荐）

- **输出格式**：
  - OGG Opus（推荐，压缩率高，音质好）
  - MP3（兼容性好）
  - AAC（现代格式）

- **比特率**：选择转码比特率（128kbps、192kbps、256kbps 等）

- **Режим транскодирования**:
  - **Никогда**: не транскодировать, передавать исходный файл напрямую
  - **Всегда**: всегда транскодировать в указанный формат
  - **Если формат не поддерживается**: транскодировать только если браузер не поддерживает исходный формат (рекомендуется)

- **Формат вывода**:
  - OGG Opus (рекомендуется, высокая степень сжатия, хорошее качество)
  - MP3 (хорошая совместимость)
  - AAC (современный формат)

- **Битрейт**: выбор битрейта транскодирования (128kbps, 192kbps, 256kbps и т.д.)

#### ReplayGain 设置 / Настройки ReplayGain

- **ReplayGain 模式**：
  - **关闭**：不使用 ReplayGain
  - **自动**：自动选择曲目级别或专辑级别
  - **曲目级别**：使用曲目级别的 ReplayGain 值
  - **专辑级别**：使用专辑级别的 ReplayGain 值

- **前置放大增益**：调整整体音量（-15dB 到 +15dB）

- **Режим ReplayGain**:
  - **Выключено**: не использовать ReplayGain
  - **Автоматически**: автоматический выбор уровня трека или альбома
  - **Уровень трека**: использование значения ReplayGain уровня трека
  - **Уровень альбома**: использование значения ReplayGain уровня альбома

- **Предусиление**: регулировка общей громкости (-15dB до +15dB)

### 界面设置 / Настройки интерфейса

- **语言**：选择界面语言（如果支持）

- **Язык**: выбор языка интерфейса (если поддерживается)

## ❓ 常见问题 / Часто задаваемые вопросы

### 播放问题 / Проблемы с воспроизведением

#### Q: 音乐无法播放 / В: Музыка не воспроизводится

**A: 可能的原因和解决方案：**

1. **浏览器不支持音频格式**：
   - 在设置中启用转码
   - 选择浏览器支持的格式（如 OGG Opus）

2. **网络问题**：
   - 检查网络连接
   - 检查防火墙设置

3. **文件损坏**：
   - 检查音频文件是否完整
   - 尝试重新扫描媒体库

**A: Возможные причины и решения:**

1. **Браузер не поддерживает аудиоформат**:
   - Включите транскодирование в настройках
   - Выберите формат, поддерживаемый браузером (например, OGG Opus)

2. **Проблемы с сетью**:
   - Проверьте сетевое соединение
   - Проверьте настройки файрвола

3. **Повреждённый файл**:
   - Проверьте, не повреждён ли аудиофайл
   - Попробуйте повторно отсканировать медиатеку

#### Q: 播放卡顿或缓冲 / В: Заикание или буферизация при воспроизведении

**A: 解决方案：**

1. **降低转码比特率**：在设置中选择较低的比特率（如 128kbps）
2. **检查服务器性能**：确保服务器有足够的 CPU 和内存
3. **检查网络带宽**：确保网络带宽足够

**A: Решения:**

1. **Снизьте битрейт транскодирования**: выберите более низкий битрейт в настройках (например, 128kbps)
2. **Проверьте производительность сервера**: убедитесь, что у сервера достаточно CPU и памяти
3. **Проверьте пропускную способность сети**: убедитесь, что пропускная способность сети достаточна

### 扫描问题 / Проблемы со сканированием

#### Q: 扫描很慢 / В: Сканирование очень медленное

**A: 可能的原因：**

1. **音乐库很大**：这是正常的，大库需要更长时间
2. **文件系统慢**：检查磁盘 I/O 性能
3. **CPU 性能不足**：元数据提取需要 CPU 资源

**A: Возможные причины:**

1. **Большая музыкальная библиотека**: это нормально, большие библиотеки требуют больше времени
2. **Медленная файловая система**: проверьте производительность I/O диска
3. **Недостаточная производительность CPU**: извлечение метаданных требует ресурсов CPU

#### Q: 某些文件没有被扫描 / В: Некоторые файлы не отсканированы

**A: 检查以下内容：**

1. **文件扩展名**：确保文件扩展名在扫描设置中
2. **文件权限**：确保服务器有读取权限
3. **文件格式**：确保文件是有效的音频文件

**A: Проверьте следующее:**

1. **Расширение файла**: убедитесь, что расширение файла указано в настройках сканирования
2. **Права доступа к файлу**: убедитесь, что у сервера есть права на чтение
3. **Формат файла**: убедитесь, что файл является действительным аудиофайлом

### 界面问题 / Проблемы с интерфейсом

#### Q: 封面图片不显示 / В: Изображения обложек не отображаются

**A: 解决方案：**

1. **检查扫描设置**：确保启用了封面提取
2. **重新扫描**：尝试重新扫描媒体库
3. **检查文件权限**：确保服务器可以读取图像文件

**A: Решения:**

1. **Проверьте настройки сканирования**: убедитесь, что включено извлечение обложек
2. **Повторное сканирование**: попробуйте повторно отсканировать медиатеку
3. **Проверьте права доступа к файлам**: убедитесь, что сервер может читать файлы изображений

#### Q: 页面加载很慢 / В: Страницы загружаются медленно

**A: 可能的原因：**

1. **数据库性能**：大库可能需要优化
2. **网络延迟**：检查网络连接
3. **浏览器缓存**：尝试清除浏览器缓存

**A: Возможные причины:**

1. **Производительность БД**: большие библиотеки могут требовать оптимизации
2. **Задержка сети**: проверьте сетевое соединение
3. **Кэш браузера**: попробуйте очистить кэш браузера

### 账户问题 / Проблемы с учётными записями

#### Q: 忘记密码 / В: Забыт пароль

**A: 解决方案：**

1. **管理员账户**：如果是管理员，可以重置其他用户密码
2. **普通用户**：联系管理员重置密码
3. **数据库重置**：如果是唯一用户，可能需要重置数据库（谨慎操作！）

**A: Решения:**

1. **Учётная запись администратора**: если вы администратор, можете сбросить пароль других пользователей
2. **Обычный пользователь**: обратитесь к администратору для сброса пароля
3. **Сброс БД**: если вы единственный пользователь, может потребоваться сброс БД (осторожно!)
